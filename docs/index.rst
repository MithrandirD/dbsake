Welcome to dbsake
=================

.. image:: _static/sake-icon.png

dbsake is a collection of command-line tools to perform various DBA related
tasks for MySQL.

.. code-block:: bash

   # curl http://get.dbsake.net > dbsake
   # chmod u+x dbsake
   # dbsake sandbox
